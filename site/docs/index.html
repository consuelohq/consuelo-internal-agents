Consuelo SDK Documentation

Consuelo is an open source sales infrastructure SDK. It gives you AI-powered dialing, real-time coaching, call analytics, and contact management â€” as code.

Quick Install

curl -fsSL consuelohq.com/install | bash

Or install via npm:

npm install @consuelo/sdk

Hello World

import { Coach } from '@consuelo/coaching';

const coach = new Coach({ apiKey: process.env.GROQ_API_KEY });

const tips = await coach.coach([
  { role: 'customer', content: 'I'm not sure about the price...' },
  { role: 'sales_rep', content: 'I understand. Let me walk you through the value.' },
]);

console.log(tips);

Packages

Each module is independently installable:

npm install @consuelo/dialer     # Twilio-powered dialer
npm install @consuelo/coaching   # AI coaching engine
npm install @consuelo/analytics  # Call analytics + transcription
npm install @consuelo/contacts   # Contact + queue management
npm install @consuelo/api        # REST API layer
npm install @consuelo/cli        # CLI tool

Architecture

Consuelo is a monorepo of composable packages. Use the unified @consuelo/sdk or pick individual modules:

import { dialer, coaching, analytics, contacts } from '@consuelo/sdk';

Self-hosted vs Managed

Self-host with your own Twilio + LLM keys, or use Consuelo Zen for zero-config managed infrastructure.
