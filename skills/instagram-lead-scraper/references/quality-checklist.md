# Lead Quality Checklist

## Must-Have Criteria

**Profile Basics:**
- [ ] Public profile (can view posts)
- [ ] Recent post within 30 days
- [ ] Bio mentions insurance/life/final expense
- [ ] Clear profile photo (not generic)

**Business Indicators:**
- [ ] Mentions being an agent/advisor/broker
- [ ] Has website or contact info
- [ ] Professional content (not personal only)
- [ ] Uses relevant hashtags

## Scoring System

**High Value (Score: 8-10)**
- Final expense specialist
- Recent business posts (last 7 days)
- 1000+ followers
- Website/contact in bio
- Active in comments
- Mentions looking for leads/products

**Medium Value (Score: 5-7)**
- General life insurance agent
- Recent posts (last 14 days)
- 500+ followers
- Some engagement
- Professional bio

**Low Value (Score: 1-4)**
- Part-time or new agent
- Posts infrequently
- Low engagement
- Unclear specialization

**Reject (Score: 0)**
- Private profile
- No recent posts (30+ days)
- No insurance content
- Spam/fake account

## Data Collection Checklist

**Required Fields:**
- [ ] Instagram handle (@username)
- [ ] Full name (if available)
- [ ] Bio/description
- [ ] Scraped date

**Recommended Fields:**
- [ ] Location
- [ ] Follower count
- [ ] Last post date
- [ ] Contact info (email/website)
- [ ] Hashtags found
- [ ] Quality score (1-10)

**Notes Field:**
- Posting frequency
- Content quality
- Engagement level
- Any specific products mentioned

## Red Flags to Avoid

**Fake/Spam Accounts:**
- Bot-like comments
- Generic profile photo
- Following thousands, few followers
- No original content

**Not Relevant:**
- Insurance customers (not agents)
- MLM recruiters
- Non-US agents (if targeting US)
- Competitors (if applicable)

**Low Intent:**
- Inactive for 90+ days
- Personal account only
- No business indicators
